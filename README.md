# TestTask-Aspect-Simple-Calculator

### Кратко
Приложение на **Qt 5.15 / QML**, реализующее классический калькулятор с разделением логики и интерфейса по паттерну **MVP**. Поддерживает базовые арифметические операции, вычисление процентов, работу с отрицательными числами и предпросмотр результата в процессе ввода, без приоритета операций.

### Архитектура
* **View (QML)** — уровень представления: `main.qml` и `CalculatorButton.qml` отвечают за адаптивную верстку, анимации и переключение тем оформления.
* **Presenter (`CalculatorPresenter`)** — связующее звено: обрабатывает ввод пользователя, управляет состоянием (ошибки, история, ввод) и обновляет свойства для отображения в UI через систему сигналов.
* **Model (`CalculatorModel`)** — логика: хранит операнды и текущую выбранную стратегию вычислений, выполняет расчеты.
* **IArithmeticStrategy** — интерфейс для математических операций.
* **ArithmeticFactory** — фабрика для динамического создания объектов стратегий по символьному коду операции.



### Функциональные возможности
* Стандартные операции: `+`, `-`, `*`, `/`.
* Вычисление процентов.
* Предосмотр: результат вычисляется автоматически при вводе второго числа, отображая промежуточное значение в главном.
* История операций: текстовая строка сверху показывает текущую последовательность действий.
* Обработка исключений: корректный вывод сообщения "Error" при делении на ноль.
* Смена тем поддержка светлого и темного режимов оформления.
* Не поддерживает приоритет операций.
